## 자바스크립트 기초
```js
var a = 10;  // var - 재선언 O, 값변경 O
var a = 18;
document.write(a + '<br>');

let b = 10;  // let - 재선언 X, 값변경 O
b = 18;
document.write(b + '<br>');

const c = 10;  // const - 재선언 X, 값변경 X
c = 18;
document.write(c + '<br>');
```
